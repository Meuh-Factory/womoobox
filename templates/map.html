World of MooBox

About

Womoobox is a very simple backend for a Firefox app: Meuh.

Goal of this backend is to store and display the locations of every Moo generated in the world by the mobile app.

A Moo can be generated with different types of animals and is represented by different pictures (according to the selected animal) on a OpenStreet-map layer.

Each time a moo is generated, user location is sent to a database and displayed on this map. No personnal data are stored on the database.
